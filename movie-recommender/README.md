# Movie Recommendation System

## About
A content-based movie recommendation system built using Python and Machine Learning.
It suggests similar movies based on genre using TF-IDF and Cosine Similarity.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn (TF-IDF, Cosine Similarity)
- Jupyter Notebook

## How It Works
1. Movies dataset is created with titles and genres
2. TF-IDF converts genre text into numerical vectors
3. Cosine Similarity calculates how similar movies are
4. Top 3 most similar movies are recommended

## Example Output
Input: The Dark Knight
Output: The Godfather, Pulp Fiction, Fight Club
